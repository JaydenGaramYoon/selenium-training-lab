Selenium automation training lab created for mastering core QA automation skills using Java, Maven, TestNG, and WebDriver.

## `src/` Overview (Folders & Files)

### `src/basics/`
Introductory Selenium WebDriver examples and browser fundamentals.

- `SelIntroduction.java` — Basic WebDriver setup and a first end-to-end run example.
- `Locators.java` — Locator strategies (e.g., id/name/className/cssSelector/xpath) practice.
- `Locators2.java` — Additional locator exercises and variations for finding elements.
- `Locators3.java` — More locator practice (typically focused on improving reliability/readability).
- `WindowActivities.java` — Browser/window actions (navigation, window/tab-related interactions).

### `src/elements/`
Hands-on exercises for interacting with common web UI elements.

- `AutoSuggestive.java` — Auto-suggest / typeahead input handling.
- `CheckBoxes.java` — Checkbox selection and state validation.
- `CheckBoxes2.java` — Additional checkbox scenarios and variations.
- `DynamicDropDowns.java` — Dynamic dropdown handling (options loaded/changed at runtime).
- `StaticDropdown.java` — Static dropdown handling (e.g., `Select` class usage).
- `StyleOpacity.java` — Working with elements affected by styling (e.g., opacity/visibility-related behavior).
- `UpdatedDropDown.java` — Updated/alternative dropdown scenario practice.

### `src/java_basics/`
Core Java refreshers used as a foundation for writing automation code.

- `CoreJavaBrushUp1.java` — Java basics (syntax, types, loops, etc.).
- `CoreJavaBrushup2.java` — Additional Java basics (collections/conditions/loops, etc.).
- `CoreJavaBrushUp3.java` — Further Java practice (continuation/refinement).
- `MethodsDemo.java` — Method creation/calls and basic OOP practice.
- `MethodsDemo2.java` — Additional method/OOP practice (continuation).

### `src/practices/`
Mini practice scenarios that resemble real automation tasks.

- `FormSubmission.java` — End-to-end form fill + submit automation practice.
